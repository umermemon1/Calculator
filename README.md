# 🧮 Calculator App

A modern and responsive calculator built using **HTML**, **CSS**, and **JavaScript**. This calculator performs basic arithmetic operations, supports **calculation history saving**, and includes a **theme toggle** feature for switching between Light and Dark modes.

## ✨ Features

* Basic arithmetic operations

  * Addition (+)
  * Subtraction (-)
  * Multiplication (×)
  * Division (÷)
* 📜 Calculation History
* 🌗 Light/Dark Theme Toggle
* 📱 Responsive Design
* ⚡ Fast and Lightweight
* 🎨 Clean User Interface

## 🛠️ Built With

* HTML5
* CSS3
* JavaScript (ES6)

## 🚀 Getting Started

1. Clone the repository:

```bash
git clone https://github.com/umermemon1/Calculator.git
```

2. Navigate to the project folder:

```bash
cd Calculator
```

3. Open `index.html` in your browser.

## 📖 Usage

* Enter numbers using the calculator buttons.
* Select an operator (`+`, `-`, `×`, `÷`).
* Press `=` to calculate the result.
* View previous calculations in the History section.
* Toggle between Light and Dark themes using the theme switch button.
